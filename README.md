# FileRecoveryApp - Advanced Deleted File Recovery

एक advanced file recovery app jo deleted files ko byte-level carving se recover karta hai.

## Features

✅ **Bina Root ke kaam kare** (Emulated Storage Access)
✅ **Developer Mode Integration**
✅ **Byte-Level File Carving**
✅ **3-Tier Monetization Model**
✅ **In-App Adjustments & Advanced Controls**
✅ **Support for Multiple File Types** (JPEG, PNG, PDF, MP4, etc.)
✅ **Real-time Progress Tracking**
✅ **Recovered File Preview**

## 3-Tier Model

### Free Tier 🆓
- Basic recovery functionality
- 500MB partition limit
- JPEG + PNG support only
- 5 recoveries per day limit
- Limited UI controls

### Pro Tier 💎
- Full recovery (10GB limit)
- All file formats support
- Unlimited recoveries
- Advanced filters
- Priority support
- ₹99/month or ₹499/year

### Premium Tier 👑
- Unlimited recovery (No size limit)
- Custom byte signature search
- Cloud backup integration
- Advanced analytics
- 24/7 support
- Custom recovery profiles
- ₹199/month or ₹1299/year

## Architecture

```
Java/Kotlin Layer (UI)
         ↓
    JNI Bridge
         ↓
C++ NDK Layer (Byte Carving Engine)
         ↓
Raw Storage Access
```

## Requirements

- Android 7.0+ (API 24+)
- Android Studio 2023.1+
- NDK r21+
- CMake 3.22.1+
